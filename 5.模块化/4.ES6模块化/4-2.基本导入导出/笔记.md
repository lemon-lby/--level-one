ES6的模块导入导出分为两种：


# 基本导入导出

1. 基本导出

    类似于exports.xxx = xxx;

    基本导出的语法如下：

    ```js
    export 声明表达式
    ```

    或

    ```js

    export {具名符号}

    ```
    由于基本导出必须具有名称，所以要求导出内容必须跟上声明表达式或者具名符号

2. 基本导入

由于使用的是**依赖预加载**，因此导入任何其它模块，导入代码必须放置到所有代码之前

对于基本导出，如果要进行导入，使用下面的代码：

```js
import {导入的符号列表} from "模块路径"
```

注意以下细节：
- 导入时，可以通过关键字```as```对导入的符号进行重命名
- 导入时使用的符号是常量，不可修改
- 可以使用*号导入所有的基本导出，形成一个对象
